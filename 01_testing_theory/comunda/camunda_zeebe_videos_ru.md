# Видео на русском языке про Camunda и Zeebe на YouTube

## Лучшие варианты для ознакомления

### 1. Подкаст GetAnalyst — «Внедряем Camunda» (~48 минут)

**YouTube:** https://youtu.be/zxwAuPwv9Bk

Лучшее вводное видео для знакомства с Camunda. Ведущая Екатерина Ананьева беседует с системным аналитиком Кристиной Виноградовой о реальном опыте внедрения. Темы: что такое Camunda, оркестрация vs хореография микросервисов, нотация BPMN, DMN как дополнение к BPMN, инструменты моделирования, интеграция в существующие системы, рекомендации по началу работы.

### 2. BPMN2ru — «Исторические предпосылки BPM, история BPMN» (~1 час)

**YouTube:** https://www.youtube.com/watch?v=RZxeZMnQ4YA

Q&A-сессия с Денисом Котовым (BPM-архитектор Т-Банка, Camunda Champion). Час ответов на вопросы: история BPM и BPMN, что такое BPMS, подход MVP, работа со стейкхолдерами, экосистема Camunda. Денис управляет 14 000+ уникальных процессов в Т-Банке на Camunda.

### 3. BPMN2ru — «Примеры хороших диаграмм BPMN из реальной жизни»

**YouTube:** https://www.youtube.com/watch?v=pS9BSasz9L8

Разбор реальных рабочих BPMN-диаграмм из продакшн-проектов на Camunda. Примеры из банковской и корпоративной среды.

### 4. BPMN2ru — «Camunda и не-Java языки: external task на примере»

**YouTube:** https://youtu.be/r6FTyHBrCEw

Туториал по использованию Camunda с языками за пределами Java через паттерн External Task. С примерами кода и ссылкой на GitHub-репозиторий.

## Видеокурс Reunico по Camunda 8 (бесплатный)

**Компания:** Reunico (reunico.com) — сертифицированный партнёр Camunda в Центральной Азии.

Бесплатный курс из нескольких уроков «Изучаем Camunda 8 вместе с Reunico», автор — Мстислав (основатель компании). Уроки:

1. Введение в Camunda 8 — архитектура, кластер Zeebe, Brokers, Gateways, gRPC, Elasticsearch
2. Создание и деплой исполняемого процесса — Camunda Modeler, деплой в Zeebe, Operate
3. Пользовательские задачи и Camunda Forms
4. Сервисные задачи и шлюзы (требуется опыт Spring Boot)
5. DMN и оркестрация сервисов
6. Parallel Gateway и Call Activity
7. Best practices

Курс опубликован как плейлист на YouTube. Для поиска: **«Reunico Camunda 8 видеокурс»** в поиске YouTube.
